Neural network model to predict projectile (ballistic) motion from time input using noisy simulated data. 
Combines <h4>physics + machine learning </h4>. 
🧠  <h2>Key Features</h2>
📊 Physics-based trajectory simulation
🔊 Noise injection (realistic sensor behavior)
⚙️ Data normalization for stable learning
🤖 Deep Neural Network (MLP with Dropout)
📉 Loss tracking and visualization
📈 Performance evaluation (MAE & RMSE)
🎯 True vs Predicted trajectory comparison
